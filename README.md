# Jade Activate

## Install
```bash
$ cd pycli
$ python -m venv env
$ source ./env/bin/activate
$ pip install -r requirements.txt
```

## Run
```bash
$ cd ./server && yarn start
```

```bash
$ cd ./client && yarn start
# open localhost:3000 in Chrome
```

```bash
$ cd pycli
$ source ./env/bin/activate
$ mpirun -np 2 python3 main.py
```
